# Weather App

Simple app for checking current weather in any city. Using free API OpenWeatherMap.

![Скриншот приложения](screenshots/app.png)

## Features

- Search weather with city name
- Weather icon
- Refresh button

### Requirements
- Qt 5.15 or higher
- CMake 3.5+

### Build and run

```bash
cd weather-app-qt
mkdir build && cd build
cmake ..
make
./weather-app-qt
